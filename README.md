# 📊 Member Activity & Skill Dashboard (Power BI)

## 📌 Overview

This project presents a Power BI dashboard analyzing member activity levels and skill distribution.
The goal is to provide clear insights into engagement patterns and overall team strength.

---

## 📈 Key Insights

* Majority of members fall within the **2200–2599 skill range**
* **Partially active** is the most common activity level
* Higher-skilled members show different activity patterns compared to lower-skilled members
* A small portion of members have **no activity**, indicating potential engagement gaps
